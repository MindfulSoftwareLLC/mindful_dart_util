# Mindful Software Dart Util

[![License: Apache 2.0][license_badge]][license_link]

Utilities that only depend on Dart.
From [Mindful Software](https://mindfulsoftware.com)

## Installation
Add `mindful_dart_util` to your `pubspec.yaml`:

```yaml
dependencies:
  mindful_dart_util:
```

Install it:

```sh
dart pub get
```
